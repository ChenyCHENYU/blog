koa 中的 session

session 的工作流程：

当浏览器访问服务器并发送第一个请求时，服务器会创建一个 session 对象，生成一个类似于 key,value 的键值对，然后将 key(cookie)返回到浏览器(客户端)，浏览器下次再访问时，携带 key(cookie)，找到对应的 session(value),客户的信息保存在 session 中。

koa-session 的使用：

npm install koa-session --sava

今天为表达全国各族人民对抗击新冠肺炎疫情斗争牺牲烈士和逝世同胞的深切哀悼，各大网站也纷纷将界面弄成黑白以示哀悼。

实现网页黑白办法其实很简单，只需要设置 filter 滤镜为 grayscale 就可以了。

常用几个值
grayscale ：将图像转换为灰度图像。
blur：可以实现高斯模糊效果。
Saturate ：设置饱和度。
Invert : 颜色反转。
Contrast ：设置对比度。
